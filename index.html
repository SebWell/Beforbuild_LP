Questions fréquentes

BeForBuild est une plateforme SaaS unifiée pour les professionnels de l'immobilier et du BTP. Elle regroupe 3 modules complémentaires : ChantierDoc (gestion documentaire intelligente avec IA), ImmoContrat (gestion contractuelle) et ImmoRisk (suivi financier et gestion des risques). Le tout sur une seule interface, conçue par un professionnel du terrain.
BeForBuild est conçu pour tous les professionnels de l'immobilier et du BTP : promoteurs, foncières, entreprises de construction, gestionnaires de patrimoine, administrateurs de biens, AMO. Les modules sont indépendants — vous pouvez utiliser uniquement ceux qui correspondent à votre métier.
ChantierDoc intègre une IA basée sur la technologie RAG (Retrieval-Augmented Generation). Concrètement : vous posez une question en langage naturel, l'IA cherche dans tous vos documents (PDF, plans, CR, marchés) et vous répond avec les sources. Elle inclut aussi l'OCR pour indexer automatiquement vos documents scannés.
Oui. BeForBuild repose sur Supabase (PostgreSQL) pour la base de données, avec authentification sécurisée et Row Level Security — chaque utilisateur n'accède qu'à ses propres données. L'application est déployée sur Cloudflare, avec chiffrement HTTPS, headers de sécurité stricts et hébergement en Europe. Vos données restent les vôtres.
Absolument. Les 3 modules (ChantierDoc, ImmoContrat, ImmoRisk) sont indépendants. Vous pouvez commencer avec un seul et activer les autres quand vous en avez besoin. Ils partagent la même base de projets, donc les données circulent naturellement entre eux quand vous les utilisez ensemble.
